# ハシラガキ (Hashira-Gaki)

脚本・シナリオのテキストデータを、標準的な脚本形式（B5・縦書き）に変換するWebツールです。

---

## 使用する記号（メタ文字）と記述ルール

ツールが各行の役割を正しく判別するために、以下の記号（メタ文字）を必ず使用してください。

### ■ タイトル
行頭に `#`（半角ハッシュ）を入力します。
- 記述例: `# 戻れない夏`

### ■ シーン（柱）
行頭に `○`（全角丸）を入力します。
- 記述例: `○ 公園・夕方`
- 変換後: 自動でシーン番号が付与され、四角い枠で囲まれます。

### ■ セリフ
`名前` + `「` + `内容` + `」` の形式で記述します。
- 記述例: `田中「お疲れ様です」`
- 変換後: 名前が自動で4文字幅に整えられ、セリフの開始位置が統一されます。

### ■ ト書き
記号を使わずに記述します。
- 記述例: `遠くでカラスが鳴いている。`
- 変換後: 自動的に行頭が6文字分下がった状態で表示されます。

---

## 操作手順

### 1. テキストの読み込み
「テキストを読み込む」ボタンを押し、PC内にある `.txt` ファイルを選択します。

### 2. 内容の編集
画面左側のエディタ領域で、直接テキストを修正・追記できます。変更はリアルタイムで右側のプレビューに反映されます。

### 3. PDFの保存・印刷
「PDFとして保存・印刷」ボタンを押します。
印刷画面が表示されたら、プリンター（送信先）を **「PDFに保存」** に設定して保存してください。

## 管理情報
- **公開形式**: GitHub Pages
- **主要構成ファイル**: `index.html`
- **検索除外設定**: `noindex` 設定済み（検索結果には表示されません）
