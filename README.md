# ハシラガキ (Hashira-Gaki)

脚本・シナリオのテキストデータを、標準的な脚本形式（B5・縦書き）に変換するWebツールです。

### ページ規格
- **用紙サイズ**: B5（縦書き）
- **フォント**: 読みやすい明朝体を採用

## 記述ルール
ツールが各行の役割を正しく判別するために、以下の記号（メタ文字）を使用して記述してください。

| 要素 | 記号・記述形式 | 記述例 | 変換後の挙動 |
| :--- | :--- | :--- | :--- |
| **タイトル** | 行頭に `#` | `# 戻れない夏` | 表紙の中央に大きく配置されます |
| **シーン（柱）** | 行頭に `○` | `○ 公園・夕方` | 枠線で囲まれ、シーン番号が自動付与されます |
| **セリフ** | `名前「セリフ」` | `田中「こんにちは」` | 名前を4文字幅に整え、開始位置を揃えます |
| **ト書き** | なし（文章のみ） | `蝉の声が響いている。` | 自動的に行頭が6文字下がります |

## 操作手順

1. **テキストの読み込み**
   「テキストを読み込む」ボタンから .txt ファイルを選択します。
2. **編集**
   画面左側のエディタで内容を修正できます。
3. **PDF出力**
   「PDFとして保存・印刷」ボタンを押し、送信先を「PDFに保存」に設定して保存します。
